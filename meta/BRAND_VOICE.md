# Brand Voice — Bible Flashcards

## Purpose
This app is a personal discipleship tool. The voice should feel like a quiet, helpful companion — not a productivity app, not a game, not a church website.

## Core Tone
- **Calm and clear**: no exclamation points, no urgency language, no gamification ("streak!", "level up!").
- **Respectful toward Scripture**: verse text is never truncated mid-sentence in UI labels. References are always exact (e.g., "Romans 8:28", never "Rom 8").
- **Encouraging without being effusive**: affirm progress plainly ("Verse memorized." not "Amazing job! You're a rockstar!").
- **Direct**: one action per screen. Labels say exactly what the action does.

## Vocabulary

| Use | Avoid |
|---|---|
| "Verse of the week" | "Today's verse", "Daily verse" |
| "Memorized" | "Completed", "Done", "Mastered" |
| "Available verses" | "Locked", "Remaining", "To-do" |
| "Test" or "Practice" | "Quiz", "Challenge", "Game" |
| "Review" | "Grind", "Drill" |
| "Accuracy" | "Score", "Points", "XP" |
| "Audio review" | "Auto-play", "Loop mode" |
| "Dismiss" (interruption) | "Skip", "Cancel", "Close" |

## Error Messages
Plain, non-blaming, actionable.
- "No memorized verses yet. Select a verse of the week to get started."
- "No verses available in this pack."
- "Couldn't play audio. Check that audio files are installed."

## Empty States
- Memorized list empty: "You haven't memorized any verses yet. Set a verse of the week to begin."
- Available list empty: "All verses in this pack have been memorized."

## Button Labels
Verb-first, specific:
- "Set as verse of the week" (not "Select")
- "Mark as memorized" (not "Done")
- "Start test" (not "Go" or "Begin")
- "Play verse" (not "Listen" or "Audio")
