Whole-Project Walkthrough

An end-to-end tour of Bible Flashcards: choosing a verse, learning it, testing yourself, and reviewing with audio.

1

Choose a verse

From the Verses tab, the Available list is browsed and a verse from the Navigators Topical Memory System is selected, then set as the verse of the week.

Verses → Available → "Romans 8:28" → Set as verse of the week
2

Study on Home

The Home tab now shows the verse of the week prominently. Tapping the card cycles through reference-only, text-only, and both — a simple self-test rhythm.

Home → Verse of the Week card → tap to cycle states
3

Run a test

On the Test tab, Verse of the Week mode and the Type format are chosen. The verse text is typed from memory and scored.

Test → Verse of the Week → Type → Start test
Accuracy: 94%
4

Mark it memorized

Confident in the verse, "Mark as memorized" is tapped from the verse detail screen. It now appears in the Memorized list.

Verse Detail → Mark as memorized
5

Casual review with audio

From the Review tab, a handful of memorized verses are queued in Play mode. The reference is spoken, followed by a pause, then the verse text — for ESV verses, the real Crossway recording plays instead of a synthesized voice.

Review → Play → Now Playing queue
