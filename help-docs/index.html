About this app

Bible Flashcards is a personal, single-user Android app for memorizing Scripture. There are no accounts, no server, and no multi-user features — everything lives on your device. Some optional features (ESV verse lookup and audio) connect to Crossway's ESV API with your explicit, separately recorded consent. See the User Guide's FAQ for details.

Bible Flashcards is a personal project. Scripture quotations from the ESV® Bible are used by permission of Crossway — see Settings → ESV Bible in the app for the full notice.
