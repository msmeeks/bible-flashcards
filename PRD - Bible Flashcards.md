# Bible Flashcards App PRD
## Problem
I want a simple flashcard app on my android phone to help me memorize Bible verses and review the ones I have previously memorized.

## Current Phone Details
I am currently using a google pixle 9 pro that is running android 16

## Basic Functions
Pre-load the app with memory packs from the Navigators.

Use ESV, CSB or NLT Bible versions for the verses.

I want to work on memorizing 1 verse per week. 

Give me the option to see a list of all the verses I have memorized and a list of the available verses from the memory packs that I have yet to memorize.

Allow for adding new verses to the list to memorize and for selecting the next verse to memorize.

Add a test feature to help me confirm my memorization. It should have 2 modes - verse of the week and review. The verse of the week test should just test memorization for the current verse of the week. Teh review test should choose 5 previously memorized verses at random and test retention of those verses. The test can have multiple options: recite the verse, type the verse, fill in the blanks. recite and type the verse can either prompt with the verse or the reference and let the user recite or type the text or they can go the other way and prompt with the text and let the user recite or type the reference. Score each verse for accuracy and give a total score averaging each verse.

## Advanced Features
Add the ability play an audio of a verse. play the reference and pause for the amount of time it takes to play the text, then play the text.

Add an option to turn on audio review. Audio review will play the verse or the week or a random prior memorized verse using the above audio format. 

Add an option to interupt audio that has been playing for more than an hour with an audio review with 50% probability of using the verse of the week and 50% probability of using a prior memorized verse. Give the option to dismiss these interuptions from the lock screen or notification.
